         正在内蒙古赤峰市考察的习近平总书记15日下午来到喀喇沁旗河南街道马鞍山村，走进农户张国利家。张国利家四世同堂，家庭成员中有蒙古族、满族、汉族3个民族。习近平看到张国利家和睦相处、其乐融融，称赞他们是“民族团结之家”。习近平指出，内蒙古自治区历来是民族自治、民族团结的模范，我们搞民族区域自治的目的是为了推动民族团结。各民族要一起推动中华民族发展，像石榴籽一样紧紧抱在一起。
